1. Download my example data in required formats from this page:
    https://students.mimuw.edu.pl/~mw219725/mcvdata/
  2. Open loading panel by "New" button click.
  3. Choose species tree data file, and click "Load" to load data.
  4. After it you should see loaded species tree.
  5. Change tab to "Proteins data loading" and do the same for proteins data.
  6. Change tab to "Interactions loading". Select species, and choose file with interactions data for every selected species.
  7. Click "Update" button, now your interactions data are loading. After loading click "Close" to close loading panel.
  8. Select "Fungi" species on species tree, and click "Show network" button. Fungi protein-protein network will be shown.
  9. Change tab to "Affinity panel". Click "Refresh", and after then click
    "Start" button to cluster Fungi network.
  10. You will see that screen.
  11. Now you should layouting network by choosing one of Cytoscape Layouts. Select Cytoscape Layouts -> Group Attributes Layout -> clusterid.
  12. Select first cluster in left top corner.
  13. Select: sc, dm, ce from species tree and click "Casting".
